# Navigation
100%纯本地数据存储的导航门户静态页面

## 20251218新增功能

### 1. 分组内搜索功能
- **位置**: 在每个分组下方显示搜索框
- **功能**: 输入应用名称，实时过滤显示符合条件的应用
- **实现特性**:
  - 仅在组内有应用时显示搜索框
  - 搜索框为空时显示全部应用
  - 使用防抖（debounce）处理，等待用户输入后延迟300ms再执行过滤
  - 避免频繁的DOM重排，提升性能
  - 不匹配时显示"未找到匹配的应用"提示

### 2. 分组拖动排序
- **功能**: 支持拖动分组标签来重新排序
- **限制**: 
  - "最近访问"分组为默认分组，不支持拖动，始终保持在第一位
  - 其他自定义分组可自由拖动排序
- **实现特性**:
  - 拖动时显示半透明效果
  - 排序结果自动保存到本地存储
  - 支持任意位置拖动放置

### 3. 组内应用拖动排序
- **功能**: 支持拖动应用卡片来重新排序
- **特性**:
  - 每个应用卡片都支持拖动（cursor: move）
  - 拖动时显示缩放和透明效果
  - 排序结果自动保存到本地存储
  - 拖动排序不受搜索过滤的影响，始终重排整个组
